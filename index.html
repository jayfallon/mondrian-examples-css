What are we doing?

So we're going to explore how to recreate the above image using CSS and HTML. I've taken three separate approaches to this: Grid, Flexbox and Table. I could've done Float and Positioning as well but both of those are fairly simple and not required for this project.

Grid

Using the grid to recreate the image was actually fairly easy and a lot of fun. I've never used grid in a project before so I was only slightly familiar with it, but once I figured out the concepts it became as easy as designing a jigsaw puzzle and then just dropping the pieces into place. Everything fits nicely and is easy to play with once you get the hang of it. See the Mondrian Grid example for more detail.

FlexBox

With the Flexible Box Module I ran into a little bit more trouble as at first I was just going to freehand the thing without using an actual grid, but then after watching a certain video I figured out that I should be using flexbox with a grid instead. I'd been using the LostGrid as part of my PostCSS implementations for the past couple of years, but I don't think I'll be doing that any more as writing everything out makes a lot more sense. Lesson learned. With my first attempt I achieve the same kind of accuracy as I did with the grid version, but my second attempt came out a lot better.

Table

Don't count tables out, they're still the standard for displaying data, until I start doing it in a grid that is. I got my start with tabular layout and made a good life out of using them so I'm not going to knock them too hard, but if you look at the underlying HTML, there's absolutely no way that I'm going to be able to hand this over to someone and expect them to figure out what's going on.
