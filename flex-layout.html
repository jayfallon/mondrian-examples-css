Flexbox Layout Module

On this page I'm making use of flexbox to control the layout for not only the basic page structure (header, nav, main, footer, etc.) but I'm also using it to control the content.

The example above is based on Fig. 1, itself inspired by Piet Mondrian and it took a total of eleven rows based on sixteen columns generating a total of seventy cells to complete and I have to say that the rendering is quite accurate compared to my first clumsy attempt, which I based on the same ten cells used in the grid example except that I did it using three columns and some hokey HTML. I have since learned my lesson.

In this example, I've used flexbox reordering on rows #1 and #5 althought I could've used the same approach that I did with the grid reordering example where I applied the background color by group and then reordered the cells where I wanted them to be.

I also made use of calc to adjust the height proportions.
